# llvm2c

Translation of LLVM bitcode to C

## Dependencies

| Name | Version | 
| ---- | ------- |
| CMake | 2.8+ |
| LLVM | 4.0 |
| Clang | 4.0 |
| Boost library | |

## Installation

    git clone https://github.com/petrv7/llvm2c
    cd llvm2c
    mkdir build
    cd build
    cmake ..
    make
